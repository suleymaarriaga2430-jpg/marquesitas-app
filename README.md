#  Marquesitas App

✨ Aplicación web para la gestión de pedidos de marquesitas, diseñada para automatizar la atención al cliente, organizar pedidos y calcular ventas en tiempo real.

---

## 🌐 Demo en línea

🔗 https://marquesitas-app.onrender.com

👉 Accede a la aplicación directamente desde este enlace.

---

## 💡 Descripción

Marquesitas App es una plataforma web tipo dashboard inspirada en aplicaciones de delivery como Didi, que permite:

* 📋 Gestionar pedidos en tiempo real
* 🧁 Seleccionar múltiples ingredientes
* 💰 Calcular el total por pedido y ventas del día
* 👩‍💻 Automatizar la atención mediante un chatbot
* 📍 Registrar clientes (dirección y teléfono)

---

## 🚀 Funcionalidades principales

* 🔐 Sistema de login básico
* 🛒 Creación de pedidos dinámicos
* 📊 Visualización de pedidos en tiempo real
* 💵 Cálculo automático de ventas
* 🤖 Chatbot automatizado (simulación de IA)
* 🧾 Registro de clientes
